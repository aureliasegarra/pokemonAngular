## ⚙️ How to use this project <a id="howto"></a>
### Setup <a id="setup"></a>
To install the dependencies of the project, just run npm install command :
```
npm install
```

### Launch server <a id="launch"></a>
```bash
ng serve
```

### Display in the browser  <a id="display"></a>
👉   http://localhost:4200
